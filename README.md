# Prismatic

**The Living Manuscript and Production System**

This repository is the shared workspace for *Prismatic*, a prestige-format sci-fi novel.

## Key Files

| File | Purpose |
|---|---|
| `Prismatic.md` | The canonical manuscript (source of truth) |
| `continuity_tracker.md` | The bible — seed pod, character wounds, dimensional travel rules, open threads |
| `retool_roadmap.md` | Chapter-by-chapter retool order and status |
| `Prismatic_Story_Scaffold.md` | Full story architecture for Parts 1-4 |
| `claude_revised_instructions.md` | Claude's project instructions |
| `staging_errors.md` | Physical staging flags across all chapters |
| `Prismatic_Full_Editorial_Report.md` | Complete editorial report |

## The Three-Person System

- **Pax (Manus):** The architect. Designs chapter structure, holds continuity, reviews output.
- **Claude:** The writer. Takes chapter briefs and writes the prose.
- **Mike:** The soul. The human gut-check. The final word.

## Rules

- `Prismatic.md` is updated by Pax after each approved chapter rewrite.
- `continuity_tracker.md` is updated after every chapter change.
- No chapter is merged without passing Pax's three-pass editorial review AND Mike's approval.
